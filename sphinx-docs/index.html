adm6 - documentation¶

whats up here?¶

adm6 is a python scripting solution to generate filterscripts for a given set of IPv6 filtering devices. ip6tables is ready to use, pf.conf started to be implemented, ipf, netsh advFirewall and the like are on the wishlist.
